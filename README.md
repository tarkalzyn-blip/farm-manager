# 🐄 مزرعة الأمل — نظام الإدارة المتكامل

نظام إدارة مزرعة احترافي مبني بـ React + Firebase + Electron.

---

## 🚀 تشغيل التطبيق

### للمتصفح (تطوير):
```bash
cd farm-manager
npm run dev
# يفتح على: http://localhost:5173
```

### تطبيق الكمبيوتر (.exe):
```
dist-electron\مزرعة-الأمل-win32-x64\مزرعة-الأمل.exe
```

---

## 🛠️ بناء تطبيق جديد

```bash
# بناء تطبيق الكمبيوتر (exe)
npm run electron:build

# الناتج في:
# dist-electron\مزرعة-الأمل-win32-x64\
```

## 🔧 التقنيات

- **React 18** + Vite 6
- **Firebase Firestore** (حفظ دائم)
- **Firebase Auth** (تسجيل الدخول)
- **Chart.js** (رسوم بيانية)
- **Electron** (تطبيق مكتبي)
